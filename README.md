# MEAN E-commerce Application

A full-stack eCommerce web application built using the MEAN stack:
MongoDB, Express.js, Angular, and Node.js.

## Features
- User authentication (JWT)
- Product listing and management
- Shopping cart and checkout
- Admin dashboard

## Tech Stack
- Frontend: Angular
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT

## Project Structure
- `/client`: Angular frontend
- `/server`: Express backend

## Status
Work in progress.
